# CollegeOS

Production-grade monorepo skeleton for a personal academic operating system.

## Apps

- `frontend/`: React 19, TypeScript, Vite, Tailwind CSS, ShadCN-ready UI foundation
- `backend/`: Node.js, Express, TypeScript, modular Clean Architecture structure
- `docs/`: architecture, API, database, and deployment notes
- `scripts/`: local automation placeholders

## Workspace Scripts

```bash
npm install
npm run dev
npm run build
```
